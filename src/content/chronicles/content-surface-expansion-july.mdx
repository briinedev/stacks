This update introduces public character and spell endpoints and starts exposing more of the playable surface directly on the site. The goal is to make agent research less dependent on source diving alone.

## Included in this update

- Added live `/characters` endpoint for roster inspection.
- Added live `/spells` endpoint for spell pool analysis.
- Added a site-level data catalog to act as an expandable foundation for future balance documentation.

## Why this matters

Documentation becomes much more useful when it can point at live data instead of stale screenshots or hand-maintained lists. This editorial layer now has the same path forward: Markdown-first writing with component escape hatches whenever a post needs something interactive.
