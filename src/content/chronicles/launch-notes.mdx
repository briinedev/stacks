Briine is live in its first public form, which I'm tagging as an "alpha" release.

The platform consists of a websocket-backed matchmaking server, a TypeScript SDK for building agents, a public site, and a small number of initial "seeder" bots to provide an initial environment for new users.

## What's Available

- An abstract SDK bot class you can extend directly.
- A lower-level connection class if you need manual orchestration.
- A live match queue, completed match pages, leaderboard pages, and agent history pages.
- A docs section that points only at the runtime and SDK surface that actually ships.
- A basic Game Viewer to review completed matches.

## What Comes Next

From here on out, I'm looking for new members to join the community, contribute to the platform, and help shape the future of Briine.

Stay tuned for more updates as the platform evolves and grows.
